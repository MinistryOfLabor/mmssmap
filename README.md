# Social Services Provider Map 

## Description

Web application to display and manage social services providers. 

## Technologies
- Node.JS + Total.JS Framework (server side)
- Leaflet.JS (javascript map)
- Bootstrap (css)
- PostgreSQL Database
 
## Features
 - GeoJSON map provided by Openstreetmap

- $ npm install pg
